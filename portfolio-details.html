Project Overview

This project involved building a comprehensive e-commerce platform from the ground up for ASU Company. The goal was to create a seamless, scalable online store capable of handling thousands of products and hundreds of concurrent users.

The platform features a fully responsive frontend built with modern HTML and CSS, a robust PHP backend, and a normalised MySQL database. The admin dashboard allows complete inventory control, order management, and real-time sales analytics.

The project was completed in 6 weeks and delivered ahead of schedule, with full handover documentation and post-launch support.

Key Features

  • Product catalogue with category filtering and advanced search
  • User authentication with session management and role-based access
  • Shopping cart with persistent state across sessions
  • Secure checkout flow with order confirmation emails
  • Real-time inventory tracking and low-stock alerts
  • Admin dashboard with sales reports and analytics
  • Mobile-responsive design across all screen sizes
  • SEO-optimised product pages with structured data

Technologies Used

This project used a carefully chosen stack focused on reliability, performance and ease of maintenance for the client's in-house team.

PHP 8.1 MySQL 8 HTML5 CSS3 JavaScript ES6+ Bootstrap 5 AWS EC2 Apache Git Composer

Challenges & Solutions

One of the key challenges was handling concurrent inventory updates during flash-sale events. We solved this using MySQL transactions with row-level locking to prevent overselling.

Another challenge was optimising query performance as the product catalogue grew. We introduced indexed views, query caching and pagination, reducing average page load time by 65%.

The client also needed the system to be maintainable by a non-technical team, which led us to build a highly intuitive admin panel with contextual help tooltips and comprehensive documentation.
